1. ALT-Insert - вызвать окно генерирования конструкторов, геттеров, сеттеров
2. sout + tab - System out
3 soutv + tab - распечататать последнюю переменную
3. Ctrl + Alt + L - отформатировать текст, проставить пробелы!
4. Ctrl + space - подсказака имени переменной
5. Ctrl-P в скобках метода - показать доступные варианты аргументов метода
6. Ctrl-H - курсор на названии класса, показывает дерево наследования
7. Shift-F6 - переименовать все вхождения данной переменной
8. Shift 2 раза - поиск среди всех классов Java
